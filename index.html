Sådan bruger du siden 🚴‍♂️🌦️

  • Upload: Din GPX-rute via knappen.
  • Tidsplan: Vælg dato, starttidspunkt og forventet gns. fart eller forventet sluttidspunkt.
  • Se vejrudsigten: Systemet beregner automatisk vejret ved start og mål på ruten. Du kan selv tilføje flere punkter, f.eks. ved et depot. Tryk på kortet og punktet bliver tilføjet oversigten.
